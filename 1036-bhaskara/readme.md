## FÓRMULA DE BHASKARA

Leia 3 valores de ponto flutuante e efetue o cálculo das raizer da equação de Bhaskara. Se não for possível calcular as raízes, mostre a mensagem correspontende "***Impossível calcular***", caso haja uma divisão por 0 ou raiz de numero negativo.

### Entrada

Leia três valores de ponto flutuante (double) A, B, C.

### Saída

Se não houver possibilidade de calcular as raízes, apresente a mensagem "Impossível calcular". Caso contrário imprime o resultado das raízes com 5 digitos após o ponto, com uma mensagem correspondente conforme exemplo abaixo. Imprima sempre o final de linha após cada mensagem.

|**Exemplos de Entrada**|**Exemplos de Saída**  |
|   :---            |   :---            |
|10.0 20.1 5.1      |R1 = -0.29788      |
|                   |R2 = -1.71212      |
|                   |                   |
|0.0 20.0 5.0       |Impossivel calcular|
|                   |                   |
|10.3 203.0 5.0     |R1 = -0.02466      |
|                   |R2 = -19.68408     |
|                   |                   |
|10.0 3.0 5.0       |Impossivel calcular|
